# Big Data

Big data refers to extremely large and complex set of data that cannot be easily managed, processed or analysed using traditional data processing tools and methods. These data sets are characterised by the 3Vs: volume, velocity and variety. 

## The 3Vs of big data

## Volume

- Refers to the immense amount of data generated every second from various sources like social media, sensors, transactions and more
- Traditional databases and tools are inadequate to handle this volume of data

## Velocity

- Describes the speed at which data is generated and collected, often in real-time or near real-time


## Variety

- Represents the diverse types of data that are generated, including structured, unstructured and semi-structured data
