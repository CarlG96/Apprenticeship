# Database Structure

## Field

A field is a where a particular type of data is stored in a record, eg data for id, firstName, lastName etc

## Column

The name of the particular attribute eg id, firstName, lastName etc

## Row

Items horizontally

## Records

Data structure which is a collection of fields (elements), each with its own name and type

## Table

In an RDMS (Relational Database Management System), a data structure characterized by rows and columns, with data occupying or potentially occupying each cell formed by a row-column intersection. The table is the underlying structure of a relation.

## Variable

A variable name is the name of the label that we give to a column
