# What a network is

At its core, creating a network allows data to move. It allows multiple devices to communicate with eachother.

# What a network needs to function

- Devices that need to send or receive data (DTE - Data Terminal Equipment)
- Intermediate network equipment - creating the network (DCE - Data Communication Equipment)

At home you will have a:

- Connection to your ISP (Internet Service Provider)
- A SOHO (Small Office Home Office) Router - A device that has multiple device/ functions in 1 device
ie it's a Router, Switch, Wi-Fi Access Point, Firewall
- Devices, wired and wireless
