# Network Infrastructure

Network infrastructure contains three categories of hardware components:

1) End devices (aka hosts)
Eg Desktop computer, laptop, printer, server

2) Intermediate devices
Eg router, switch, access point, firewall

3) Network media 
Eg copper, fibre, wireless

A basic network between two computers could look like:

End device - Network Media - End device

Or scaled up ie:

End device - Network media - Intermediate Device - Network Media - End device
