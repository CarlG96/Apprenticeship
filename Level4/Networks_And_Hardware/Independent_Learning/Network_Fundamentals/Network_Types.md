# Network Types

## Subnet

The internet is actually an abbreviation for Interconnected Network

A subnet is a sub-network. It defines a local border for addresses Ie a routing border for communication

These "borders" allow:

- Traffic control - Subnets are noisy, lots of broadcasts, we want to keep these local.
- Security - Traffic segregation - a Zero Trust ideal
- Traffic shaping - to allow quality of service

## PAN - Personal Area Network 
eg Bluetooth connection between two devices

## LAN - Local area network 

A local office network or home (sometimes called a HAN). Used to be defined by one subnet, but dated bu the advent of VLANs ie you can have more than one subset locally. WLAN stands for Wireless LAN.

## CAN - Campus Area Network
Eg a university campsu with multiple buildings ie different subnets

## MAN - Metropolitan Area Network 
Eg 2 office buildings connected together in a locality
Also a WMAN - Wireless MAN - Wi-fi hotspots in towns/ cities

## WAN - Wireless Area Network 
Eg the internet, 2 or more office connected globally
