# Transistor

Transistors largely act as 'switches'. When given current, they can open and close. For a point,
they can also be 'slightly' open, meaning they can be used as amplifiers in certain applications.
Transistors are semiconductors, meaning they are neither good conductors or bad conductors. Silicon
is an example of a semiconductor.

A transistor has three terminals: a collector, an emitter, and a base. In a citrcuit, it can act as
an insulator, preventing current flowing between the collector and emitter terminals. Applying a small
current to the base terminal will allow current to flow from the collector to the emitter. Basically
a non mechanical switch

## Simple Gate

When the input is 0 the output is 0
When the input is 1 the output is 1

## Not gate

When the input is 0 the output is 1
When the input is 1 the output is 0

## AND gate (using simple gates) 

Transistors are in series
When both inputs are 0, output is 0
When one input is 0, output is 0
When both inputs are 1, output is 1

## OR gate 

Transistors are in parallel

When either gate input is 1, output is 1

## Adder

Adders are combinations of logic gates that produce a sum and carry, that allows
for addition of binary numbers. For each bit in a number, you need another adder.

# 8 bit adder

A component that can add two 8 bit numbers together, has an overflow output that
can then lead to another 8 bit adder to handle numbers that increase above 8-bit.
