# What is a bit?

Data moving in a network is moving as 0s and 1s also known as binary digits.
Byte = 8 bits
Megabyte = 1048576 bytes or 8388608 bits
lower case b is a bit and upper case B represents a byte

## Mediums

How bits are respresented through different mediums:

Wired - copper
Fibre - light intensity
Wirelessly - Frequency

## Bandwidth

Bandwidth is the term to describe the speed at which these bits moves through a medium.
For example, 900 Mbps would be 900 million bits per second. Thoroughput is another term,
it is like bandwidth but a more accurate representation of actual speed - it takes into
account the amount of data, type and other issues that the data might hit on its travels
from point A to point B.
