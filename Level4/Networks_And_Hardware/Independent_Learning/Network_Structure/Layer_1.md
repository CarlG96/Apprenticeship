# Layer 1

## Cables

Ethernet cable is the most commonly used type

## Network Card

The network card has quite a basic job on the scale of things. It is responsible for converting the computer based data to what ever media it is trying to interface with to send to or vice versa when receiving data eg

- Ethernet Network Card - Converting to copper cabling
- Wireless Network Card - Converting to a radio wave signal

## Repeater

This is used to extend the range of a media type

For Ethernet cable eg Cat 5e the maximum length of a cable is 100m. A repeater can be used to extend this. It in essence repeats the signal onwards.

For wireless, this is normally called a wifi extender.
