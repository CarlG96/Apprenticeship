# Layer 5 - Application

The Application layer is where the software and services that send and receive data reside. It is responsible for translating, encoding/decoding, encrypting/decrypting the data.

From a networking point of view, we have little control over this as it is dictated to us depending on the application.

The tricky part of this is if we want to control or monitor the data that is being sent. A Firewall, depending on it functionality, can operating at these different layers. So at Layer 5, it is wanting to inspect the actual data e.g. for content control.
